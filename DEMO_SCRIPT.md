# Memory Shield — 90s demo script (Jul 5 submission)

Pre-flight: `uvicorn memory_shield.api:app --port 8000` from `backend/`, `npm run dev` from `frontend/`. Open `/studio` once to warm caches.

## Script

1. **Hook (10s)** — "Memory that gets sharper, not just bigger. Every suggestion cites the videos that justify it."
2. **Proof strip (15s)** — Today tab: point at citation thumbnails above the fold on a concept card. "Your video at 2.2× median + competitor proof."
3. **Outliers (10s)** — Strong fit badge → **ride** the wave.
4. **Graph (15s)** — Click card → Memory drawer → Trend → Topic → Video → Format path.
5. **Skeptic stack (20s)** — Footer: **RAG contrast** (generic vs cited concept) → **sealed backtest** → break seal → 24.6× champion.
6. **Workflow (15s)** — **+ plan it** → Board → open **shoot brief** → **copy film-kit**.
7. **Close (5s)** — "Post it — memory sharpens from your uploads automatically."

## What to emphasize for judges

- Four Cognee ops: remember / recall / improve / forget
- Killer query is orchestrated graph join, not one `search()` call
- Anti-slop: proof strip + validated citations
